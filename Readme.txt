ЗАПИСНАЯ КНИЖКА v1.0

Веб-приложение для хранения контактных данных пользователей: имени, фамилии, номера телефона. Приложение позволяет добавлять, удалять, редактировать данные пользователей, а также производить контекстный поиск по одному или нескольким параметрам.


ТРЕБОВАНИЯ
Для установки и использования приложения необходимо установить JRE не ниже 7.0 версии и Apache Maven.


СБОРКА
Для сборки приложения нужно открыть командную строку, перейти в корневой каталог проекта и ввести:

mvn clean package

После этого war-файл web-app.war можно будет найти в папке target корневого каталога и станет возможен запуск приложения с помощью командной строки.


ЗАПУСК
Чтобы запустить проект, в командной строке перейдите в корневой каталог проекта и введите:

mvn exec:java

При запуске генерируется база данных и заполняется тестовыми данными.
После этого приложение доступно в браузере по ссылке:

http://localhost:8080/notebook


АВТОМАТИЧЕСКАЯ СБОРКА И ЗАПУСК(только для Windows)
Запустить run.bat

Приложение доступно в браузере по ссылке:

http://localhost:8080/

ОСТАНОВКА
Для остановки приложения - Ctrl+C
